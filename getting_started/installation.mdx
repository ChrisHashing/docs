# WTTP Installation Guide

Welcome to WTTP! Below are the steps to get started with WTTP and use it to fetch data from blockchain addresses.

---

## Installation

To install wttp-handler, you can use npm (Node Package Manager) or yarn (alternative package manager).

### Using npm

```bash
npm install wttp-handler 
```

### Using yarn

```bash
yarn add wttp-handler
```

You can now start using WTTP to interact with blockchain addresses and fetch data seamlessly. For more details on usage, refer to the Quick Start guide.
